# 🧠 Section 5: Social Engineering Attacks

## ❓ Cue / Key Questions & Notes


| Question / Keyword | Answer |
| --- | --- |
| **🟢 Social Engineering Basics** |  |
| 🔴 What is social engineering? | Manipulating individuals into revealing confidential information or performing insecure actions. |
| 🔴 What are the core principles of influence? | Human psychological triggers: Authority, Urgency, Social Proof (Consensus), Scarcity, Familiarity/Liking, Fear, and Trust. |
| 🔴 What is pretexting? | Inventing a fabricated, highly believable scenario to establish trust and trick a victim into compliance. |
| 🔴 What is impersonation? | Explicitly pretending to be someone else, such as an IT technician, trusted vendor, law enforcement, or corporate executive. |
| **🟢 Phishing Variants** |  |
| 🔴 What is phishing? | A broad, un-targeted fraudulent email campaign used to steal user credentials, collect data, or deliver malicious payloads. |
| 🔴 What is spear phishing? | A highly targeted phishing attack customized for a specific individual, team, or organization based on prior research. |
| 🔴 What is whaling? | A high-stakes spear phishing attack targeted specifically at C-level executives, politicians, or high-value individuals. |
| 🔴 What is vishing? | Voice phishing conducted over phone lines or VoIP systems to extract personal or financial credentials. |
| 🔴 What is smishing? | Phishing attacks delivered directly via SMS text messages. |
| **🟢 Targeted & Architectural Attacks** | BEC |
| 🔴 What is Business Email Compromise? | A targeted attack where an adversary compromises or spoofs an executive's email account to trick finance teams into unauthorized wire transfers. |
| 🔴 What is pharming? | Redirecting users from legitimate websites to fraudulent replicas by manipulating DNS records or local host files. |
| 🔴 What is a watering hole attack? | Compromising a specific third-party website known to be frequently visited by the target group or organization. |
| 🔴 What is typosquatting? | Registering intentionally misspelled variations of popular domain names to capture mistyped web traffic (URL hijacking). |
| 🔴 What is brand impersonation? | Mimicking the exact layout, logos, and visual identity of a trusted organization or brand to deceive victims. |
| **🟢 Influence & Fraud Operations** |  |
| 🔴 Disinformation vs. misinformation | Disinformation is deliberately false information designed to deceive; misinformation is false information spread unintentionally. |
| 🔴 What is an influence campaign? | A coordinated, often state-sponsored operation designed to manipulate public opinion, disrupt institutions, or alter social consensus. |
| 🔴 What are diversion theft and hoaxes? | Diversion theft misdirects cargo deliveries; hoaxes invent fake security warnings to trick users into altering system configurations or causing disruption. |
| 🔴 What are corporate frauds and scams? | Deceptive criminal schemes like identity theft, fraudulent invoices, or advance-fee scams designed to siphon corporate funds. |
| **🟢 Social Engineering Defenses** | MFA |
| 🔴 What are the primary defenses? | Continuous security awareness training, out-of-band verification procedures, user reporting channels, and enforcing Multi-Factor Authentication. |


## 📝 Summary


**Social engineering** shifts the cyber attack vector away from pure technical software bugs and focuses directly on human psychology, manipulating victims through core principles of influence like **authority**, **urgency**, **scarcity**, and **fear**. Adversaries execute these schemes using **impersonation** and **pretexting**, deploying them primarily through massive **phishing** operations or its localized variants: targeted **spear phishing**, executive-level **whaling**, voice-based **vishing**, and SMS-driven **smishing**. On a corporate level, these vectors expand into **Business Email Compromise (BEC)**, infrastructure manipulation like **pharming** and **typosquatting**, and strategic **watering hole attacks**. Wider operations utilize weaponized **disinformation** within coordinated, state-backed **influence campaigns** alongside resource-diverting **hoaxes**. Neutralizing these psychological attacks requires an active defensive layer built on structural **security awareness training**, out-of-band verification policies, and structural technical controls like **Multi-Factor Authentication (MFA)**.
