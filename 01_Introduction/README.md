# Section 1: Introduction


> Cornell Notes — CompTIA Security+ (SY0-701)


## About the Certification


| Question / Keyword | Answer |
| ------------------ | ------ |
| What is CompTIA Security+? | An intermediate-level IT / entry-level cybersecurity certification focused on assessing the security posture of an enterprise environment. |
| Who is it designed for? | IT professionals or aspiring cybersecurity professionals; recommended prereqs are CompTIA A+ and Network+. |
| Do I need A+/Network+ first? | No — it's a recommendation, not a strict requirement. Equivalent of 1–2 years of hands-on hardware, software, and network experience is sufficient. |
| How many exam domains? | 5 domains. |
| **Domain 1** | General Security Concepts — **12%** |
| **Domain 2** | Threats, Vulnerabilities, and Mitigations — **22%** |
| **Domain 3** | Security Architecture — **18%** |
| **Domain 4** | Security Operations — **28%** |
| **Domain 5** | Security Program Management and Oversight — **20%** |


## Summary


The CompTIA Security+ (SY0-701) is a vendor-neutral, entry-level cybersecurity certification with **90 minutes** to answer **up to 90 questions** (multiple-choice, multiple-select, and performance-based). The passing score is **750/900**. The exam covers **5 weighted domains** (General Security Concepts 12%, Threats/Vulnerabilities/Mitigations 22%, Security Architecture 18%, Security Operations 28%, Security Program Management 20%). Success requires **understanding concepts**, not memorization — skip tough questions, use the brain-dump whiteboard, and always guess rather than leave answers blank.
