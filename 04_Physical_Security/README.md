# 🚧 Section 4: Physical Security

## ❓ Cue / Key Questions & Notes


| Question / Keyword | Answer |
| --- | --- |
| **🟢 Physical Security** |  |
| 🔴 What is physical security? | Protecting tangible assets, facilities, equipment, and personnel from physical threats or environmental hazards. |
| **🟢 Perimeter Barriers** |  |
| 🔴 What are fencing and bollards? | Perimeter defenses that deter and delay intruders; bollards specifically stop vehicle ramming attacks. |
| 🔴 What is the role of physical lighting? | Acts as a deterrent control while increasing the visibility and effectiveness of video surveillance. |
| 🔴 What control type is signage? | A deterrent control used to communicate rules, boundary warnings, and legal hazards to intruders. |
| **🟢 Physical Access Controls** |  |
| 🔴 What is an access control vestibule? | Formally known as a mantrap; a space with two interlocking doors designed to explicitly prevent tailgating. |
| 🔴 Tailgating vs. piggybacking | Tailgating is following someone into a secure area *without* their consent; piggybacking is doing so *with* their consent. |
| 🔴 What is physical brute force? | Forcing entry into a facility by destroying physical barriers (e.g., breaking locks, breaching doors). |
| 🔴 How do access badges operate? | Use RFID or embedded smart card chips to provide identification and electronic authentication at entry points. |
| **🟢 Locking Mechanisms** |  |
| 🔴 What are the main door lock types? | Padlock, pin-and-tumbler, electronic, cipher (keypad), biometric, and physical cabinet locks. |
| **🟢 Biometrics** |  |
| 🔴 What traits are used in biometrics? | Fingerprint, retina, iris, facial structure, voice, vein layout, and gait (walking pattern) recognition. |
| 🔴 What are FAR and FRR? | False Acceptance Rate (unauthorized user accepted) and False Rejection Rate (legitimate user rejected). |
| 🔴 What is the Crossover Error Rate? | CER; the exact operational point where $FAR = FRR$. A lower CER signifies a much more accurate biometric system. |
| **🟢 Surveillance & Monitoring** | CCTV |
| 🔴 What control type is video surveillance? | Primarily a detective control used to record incidents, but it also offers passive deterrence. |
| 🔴 What role do security guards fulfill? | An operational/physical control providing active human deterrence, manual authentication, and incident response. |
| 🔴 What are modern automated patrol tools? | Drones and robot sentries deployed to monitor large, complex, or dangerous perimeters. |
| **🟢 Intrusion Detection Sensors** |  |
| 🔴 What sensors detect active entry? | Infrared, pressure pads, microwave frequencies, and ultrasonic soundwaves. |
| 🔴 What are the primary physical alarms? | Circuit alarms (doors/windows), motion-detection alarms, and hidden duress (panic) alarms. |


## 📝 Summary


Physical security safeguards an organization's tangible assets, personnel, and facilities through layered, defense-in-depth controls. Outer **perimeter barriers** such as fencing, heavy **bollards**, industrial lighting, and protective **signage** serve to deter and delay external threats. Internal entryways rely on electronic **access badges**, manual **locks**, and **access control vestibules** (mantraps) to mitigate the risks of physical **brute force** and unauthorized tailgating or piggybacking. Active environments are continuously audited via **surveillance** tools like **CCTV**, security guards, automated **drones**, and specialized **intrusion detection sensors**. When deployment calls for **biometrics**, systems are tuned to balance user friction and false metrics, measuring overall system accuracy via the Crossover Error Rate (CER), where $FAR = FRR$.
